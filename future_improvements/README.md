## 🔧 Troubleshooting

### Backup Before Installing
Each script automatically creates backups:
```
app.py.backup_YYYYMMDD_HHMMSS
```

### Restore Backup
```bash
cp app.py.backup_20260112_220000 app.py
```

### Script Fails?
1. Check you're in correct directory
2. Check file permissions: `chmod +x *.sh`
3. Read error message
4. Manual install (see docs/)
